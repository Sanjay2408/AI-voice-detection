## 📌 Overview

The AI Voice Detection API analyzes audio samples and determines whether the voice is **synthetically generated by AI** or **naturally spoken by a human**. It uses a trained Random Forest classifier that extracts comprehensive audio features (MFCCs, spectral contrast, chroma, mel spectrogram, tonnetz) to make accurate predictions.

### ✨ Key Features

- 🔍 **AI vs Human Classification** — Accurately detects synthetic voices  
- 📊 **Confidence Scoring** — Returns prediction confidence percentage  
- 🎧 **Multi-Format Support** — Accepts MP3, WAV, M4A, and more via base64 encoding  
- 🔐 **API Key Authentication** — Secured endpoints with `X-API-KEY` header  
- ⚡ **Fast Inference** — Real-time processing with millisecond-level response times  
- 🐳 **Docker Ready** — Containerized for easy deployment  
- 🚂 **Railway / Vercel Compatible** — Deploy to cloud in minutes

## 🚀 Getting Started

### Prerequisites

- **Python 3.11+**
- **FFmpeg** (required for audio processing)
- **Git**

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/YOUR_USERNAME/voice-detection-api.git
cd voice-detection-api
```

2. **Create a virtual environment**

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate
```

3. **Install dependencies**

```bash
pip install -r requirements.txt
```

4. **Set up environment variables**

Create a `.env` file in the project root:

```env
API_KEY=your_secret_api_key_here
```

5. **Run the server**

```bash
uvicorn app.main:app --reload --port 8000
```

The API will be available at `http://localhost:8000`

## 📡 API Reference

### `GET /`

Health check and welcome endpoint.

**Response:**

```json
{
  "message": "Welcome to the AI Voice Detection API",
  "documentation": "/docs",
  "status": "operational",
  "version": "1.0.0"
}
```

### `POST /detect-voice`

Classify an audio sample as AI-generated or human-generated.

**Headers:**

| Header      | Type     | Required | Description          |
|-------------|----------|----------|----------------------|
| `X-API-KEY` | `string` | ✅ Yes   | Your API key         |

**Request Body:**

```json
{
  "audioBase64": "<base64_encoded_audio_string>",
  "audioFormat": "mp3",
  "language": "en"
}
```

| Field         | Type     | Required | Default | Description                          |
|---------------|----------|----------|---------|--------------------------------------|
| `audioBase64` | `string` | ✅ Yes   | —       | Base64-encoded audio data            |
| `audioFormat` | `string` | ❌ No    | `mp3`   | Audio format (mp3, wav, m4a, etc.)   |
| `language`    | `string` | ❌ No    | `en`    | Language code                        |

**Response:**

```json
{
  "classification": "HUMAN_GENERATED",
  "confidence": 0.92,
  "explanation": "Spectral variance and harmonic structure are consistent with natural human speech.",
  "processing_time_ms": 134
}
```

| Field                | Type     | Description                                          |
|----------------------|----------|------------------------------------------------------|
| `classification`     | `string` | `AI_GENERATED`, `HUMAN_GENERATED`, or `ERROR`        |
| `confidence`         | `float`  | Confidence score between 0.0 and 1.0                 |
| `explanation`        | `string` | Human-readable reasoning for the classification      |
| `processing_time_ms` | `int`    | Time taken for inference in milliseconds             |

## 🧪 Example Usage

### Using cURL

```bash
# Encode your audio file to base64
base64 -i your_audio.mp3 -o audio_encoded.txt

# Make the API request
curl -X POST http://localhost:8000/detect-voice \
  -H "Content-Type: application/json" \
  -H "X-API-KEY: your_secret_api_key_here" \
  -d '{
    "audioBase64": "'$(cat audio_encoded.txt)'",
    "audioFormat": "mp3"
  }'
```

### Using Python

```python
import base64
import requests

# Read and encode audio file
with open("sample.mp3", "rb") as f:
    audio_b64 = base64.b64encode(f.read()).decode()

# Send request
response = requests.post(
    "http://localhost:8000/detect-voice",
    headers={"X-API-KEY": "your_secret_api_key_here"},
    json={
        "audioBase64": audio_b64,
        "audioFormat": "mp3"
    }
)

print(response.json())
```

## 🧠 How It Works

The API uses a **Random Forest Classifier** trained on a dataset of real and AI-generated voice samples. The ML pipeline:

1. **Audio Ingestion** — Accepts base64-encoded audio, decodes it using `pydub`  
2. **Feature Extraction** — Extracts 200+ audio features including:
   - **MFCCs** (40 coefficients — mean & standard deviation)
   - **Chroma STFT** (pitch class profiles)
   - **Mel Spectrogram** (perceptual frequency analysis)
   - **Spectral Contrast** (peak-valley differences in spectrum)
   - **Tonnetz** (tonal centroid features)
3. **Classification** — Feeds features into the trained model for prediction  
4. **Fallback** — If the model fails to load, a heuristic based on silence ratio analysis is used
